Pick of the week

Bun 1.3.6

People love to criticize JavaScript, but it's a great language for general purpose coding. Specifically, the modern ES module syntax is very ergonomic, and the npm package ecosystem is hard to compete with.

BunJS is a modern javascript runtime written in Zig that has a very developer-friendly standard library, is compatible with most NodeJS and Web APIs, and is surprisingly performant.

I have been using Bun for my projects for years, and I highly recommend giving it a try if you use NodeJS or if you're interested in learning JavaScript.

This week, Bun 1.3.6 came out, which finally supports building virtual files . This is something I've hacked around before by writing tons of temporary files during my build process. With built-in virtual file support, I'll be able to refactor some of my build processes.
